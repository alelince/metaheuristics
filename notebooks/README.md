# Pluto notebooks

## Installation

In a Julia interpreter, run the following commands:

```julia
> import Pkg
> Pkg.add("Pluto")
```

## Usage

In a Julia interpreter, run the following commands:

```julia
> import Pluto
> Pluto.run()
```
